# own_vim_ide
操作系统为deepin linux15.5 ubuntu16.10
Make a vim ide for develop as personal

打造自己高效的文本编辑器，主要目标语言为
- c
- php
- shell
- golang
- python


## 安装步骤
1 ./init.sh
提示报错后按回车继续


## 安装内容
* you complete me 
* vim-syntax-extra for c 
* phpcomplet php补全
* vdebug 调试
* molokai 配色方案
* bash-support 写shell脚本
* ctrl-p
* tagbar
* vim-surround
* vim-repeat
* vim-abolish
* vim-vinegar  filebrowser
* vim-go  golang
* jedi-vim  python complete 
* gocode
